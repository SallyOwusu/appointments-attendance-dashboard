# Appointment Attendance Dashboard (Excel Project)

## Overview
This project is an interactive Excel dashboard analysing patient appointment attendance and non-attendance (DNA) trends. It explores patterns across different patient groups to support service improvement.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualisation
- KPI Metrics

## Key Features
- Attendance vs non-attendance analysis
- Comparison by age group
- Identification of high DNA groups
- KPI summary (Attendance Rate, Total Appointments)

## Key Insights
- Certain age groups showed higher non-attendance rates
- Attendance patterns varied across patient demographics
- Identifying DNA trends can support targeted interventions
